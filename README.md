# Bandung Profile Page

An informative and fully responsive informational landing page dedicated to showcasing the rich history, geographical insights, and beautiful tourist destinations of Bandung, Indonesia (famously known as the *Paris van Java*).

This project was built as part of the **"Belajar Dasar Pemrograman Web"** (Basic Web Programming) course submission at Dicoding Academy.

![CSS3](https://img.shields.io/badge/css3-%231572B6.svg?style=for-the-badge&logo=css3&logoColor=white)

## ✨ Features

- **Fluid Responsiveness:** Seamlessly adapts to desktop, tablet, and mobile viewing widths using advanced CSS Media Queries (`max-width: 1200px` and `768px`).
- **Sticky Top Navigation:** Features a navigation bar with a `position: sticky` attribute that remains locked to the top of the viewport for effortless internal page jumping.
- **Card-Based Layout Layout:** Organizes historical data, geographic text, and tourist destinations into clean, shadowed container cards for an elevated reading experience.
- **Informational Profile Sidebar:** Utilizes a structured HTML table inside a fixed-width `aside` sidebar element to display core facts such as establishment date, area size, and local language.
- **Elegant Typography:** Styled entirely with the modern geometric sans-serif `Quicksand` font family integrated from Google Fonts.

## 🛠️ Tech Stack

- **Markup:** HTML5 (Semantic elements like `<main>`, `<article>`, `<aside>`, `<section>`, and `<footer>`)
- **Styling:** Vanilla CSS3 (Flexbox Layouts, Box Shadows, Hover Micro-interactions, and Fluid Scaling)
- **Typography:** Google Fonts (Quicksand)
